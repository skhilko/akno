Highlights

  • CSS3 appearence effects

    Uses css transitions to reveal a dialog window.

  • No dependencies

    jQuery is supported, but not required.

  • Smart focus

    Prompt automatically captures focus upon opening a window. Here are the rules for first introduced by jQuery UI:

    • The first element within the dialog with the autofocus attribute.
    • The first tabbable element within the dialog's content.
    • The first action button.
    • The dialog's close button.
    • The dialog itself.
