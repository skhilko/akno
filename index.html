How do I use it?

General Usage

Akno doesn't have any external dependencies. Simply include akno.min.js and akno.min.css on the page and you're good to go. This will load the Akno object into the global window object for you.

    
    var akno = new Akno(document.getElementById('signupDialog'));
    // ... somewhere later
    akno.close();

jQuery

In case you want akno as a jQuery plugin, make sure to include this akno.jquery.min.js build instead of the regular one. Same styles akno.min.css are still required.

    
    $('#signupDialog').akno();
    // ... somewhere later
    $('#signupDialog').akno('close');

What APIs are available?

Options

Options object can be passed as a second argument to the Akno constructor or the first and only argument to the akno jQuery plugin.

Constructor

    
    var akno = new Akno(document.getElementById('signupDialog'), {
            effect: 'newspaper',
            header: 'Please Signup'
        });

jQuery

    
    $('#signupDialog').akno({
            effect: 'newspaper',
            header: 'Please Signup'
        });
Name Type Default Description
header String Defines the header text.
effect String scale-up Open effect for the akno.

Supported values:

  • scale-up
  • slide-in-right
  • slide-in-bottom
  • newspaper
  • fall
  • side-fall
  • sticky-top
  • flip-hor
  • sign
  • scale-down
  • just-me
  • split
  • rotate-bottom
  • rotate-left
open Boolean true Shows the akno when initialized.

Methods

open

Opens the akno.

close

Closes the akno.

Please Note: When used as a jQuery plugin, any method can be called supplying the method name to the plugin $('#dialog').akno('close');.

Events

Name Description
akno-open This event is triggered when the akno has been made visible.
All animations and transitions are completed.
akno-close This event is triggered when the akno has been hidden.
All animations and transitions are completed.
