Akno

Simple, fast and beautiful modal windows

akno /akn√≥/, n. Belarusian
1. window
  • Download
  • GitHub

Features

  • CSS transitions. Let the GPU do the heavy lifting.
  • Multiple instances at a time. Supports showing more than one akno at a time.
  • No dependencies. jQuery is supported, but not required.
  • Smart tabulation. Ensures that tabulation cycles through the elements within the akno.

Usage

Using Akno is really simple. It has a pure JavaScript API, plus an optional jQuery plugin.
